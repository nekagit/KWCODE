## Prerequisites

- Node.js (version in .cursor/project/TECH-STACK.md)
- Package manager: npm | pnpm | yarn

## Installation

1. Clone the repo.
2. Install dependencies: `npm install` (or pnpm/yarn).
3. Copy `.env.example` to `.env` and set variables.
4. Run migrations or seed if applicable.

## First run

- Dev: `npm run dev`
- Build: `npm run build`
- Test: `npm run test`

See `.cursor/setup/` for architecture, frontend, and backend setup docs.
