# AI-generated ticket suggestions

Placeholder for AI-suggested tickets. The source of truth for the Kanban and Run tab remains `.cursor/planner/tickets.md`.

## How to use

1. Run an AI or script that suggests tickets (e.g. from milestones or requirements).
2. Paste or append suggestions here for review.
3. After review, copy accepted items into `.cursor/planner/tickets.md` in the right priority section and run Sync in the app.

## Format

Keep the same format as tickets.md: `- [ ] #N Title — description — @agent1 @agent2`
