KWCode

## Description

KWCode is an AI-powered development workflow and prompt orchestration desktop application. It helps teams manage projects with predefined agents, Kanban-style tickets, features, prompts, and automated implementation runs. Built with Next.js and Tauri v2 for desktop; supports browser-only mode via Next.js API.

## Tech stack summary

- **Frontend:** Next.js 16 (App Router), React 18, shadcn/ui, Tailwind, Zustand. See .cursor/setup/frontend.json.
- **Backend:** Tauri v2 (Rust), SQLite (rusqlite), Next.js API routes (browser fallback). See .cursor/setup/backend.json.
- **Documentation:** .cursor/documentation/ and docs/
