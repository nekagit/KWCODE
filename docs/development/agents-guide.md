# Agents guide

Role definitions live in `.cursor/agents/` (e.g. backend-dev, frontend-dev, solution-architect, tester, documentation-writer).

- **Run tab:** Use "Implement All" with `.cursor/prompts/worker.md`; agents are loaded per ticket from `.cursor/agents/`.
- **Ticket generation:** When creating a ticket, assign agents; the app lists all `.md` files in `.cursor/agents/`.
