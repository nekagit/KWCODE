# Agents guide

Role definitions live in `.cursor/2. agents/` (e.g. backend-dev, frontend-dev, solution-architect, tester, documentation-writer).

- **Run tab:** Use "Implement All" with `.cursor/8. worker/implement-all.md`; agents are loaded per ticket from `.cursor/2. agents/`.
- **Ticket generation:** When creating a ticket, assign agents; the app lists all `.md` files in `.cursor/2. agents/`.
